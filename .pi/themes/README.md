# Themes

Local TUI themes live here.

## Tau Pack

- `tau-dark`: default dark terminal daily theme.
- `tau-focus`: lower contrast, fewer bright accents.
- `tau-alert`: stronger warning/error contrast for safety and debug work.

Use inside `tau ext vibe`:

```text
/theme
/theme tau-dark
/theme tau-focus
/theme tau-alert
```

Shortcuts:

- `Ctrl+Shift+T`: next theme.
- `Ctrl+Alt+T`: previous theme.
